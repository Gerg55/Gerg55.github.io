C Engine (in development)

Current features:

  • 3D first person camera control
  • .ply format model loading
  • A simple Blinn-Phong lighting model
  • Real-time directional shadow mapping
  • Console commands to modify scene (will be replaced with a GUI)

Purpose:

The point of this project is to learn all about how games work. I am challenging myself to use as
few libraries as possible (besides GLFW and GLAD). This means I have to make most things
myself such as a maths library, physics, asset loaders, GUI, etc. By the end of the project
I will hopefully have a decent platform I can make games on top of.
